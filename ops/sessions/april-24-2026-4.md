## Exit code semantics
- `0` — all checks passed
- `1` — DB present but at least one check failed
- `2` — DB file does not exist

The `info` subcommand exits 1 for a missing DB; `verify` exits 2. Intentional divergence: PR next-4's badge maps 2 → "Unknown" and 1 → "Needs attention".
