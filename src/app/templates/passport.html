Memory Passport packages your conversations with source tracking so exports can be inspected and validated — without replacing your local archive.

What Memory Passport is

Memory Passport is an export-and-validation package. It describes what was exported and where each piece originated, so you can verify your archive's integrity.

It does not replace your local archive. Your conversations remain the source of truth; the passport is an export snapshot with integrity evidence.

What export does

Export writes an inspectable package from your archive. Your conversations are copied into a structured package, while generated materials (like markdown views) are included as supplementary representations tied back to their source.

Export is currently available through the existing CLI. You can also use the Export button below.

python -m src.passport.cli export <output_dir>

What validation does

Validation checks passport package integrity and internal coherence: required files, record structure, provider consistency, and source links.

  • Valid: integrity checks passed without diagnostics.
  • Valid with warnings: core integrity passed, with non-blocking issues to review.
  • Invalid: one or more blocking integrity failures were detected.

python -m src.passport.cli validate <passport_path>
