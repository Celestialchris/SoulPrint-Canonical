Recommended: Ollama + Gemma 4 (local, free, private)

Install Ollama, then pull a model:

ollama pull gemma4           # 9.6 GB, runs on 6+ GB VRAM (recommended default)
ollama pull gemma4:26b       # 18 GB, runs on 12+ GB VRAM (better quality, 256K context)

Then set these environment variables before starting SoulPrint:

SOULPRINT_LLM_PROVIDER=openai
SOULPRINT_LLM_BASE_URL=http://localhost:11434/v1
SOULPRINT_LLM_MODEL=gemma4
Other options: cloud providers

Any OpenAI-compatible API works. Set SOULPRINT_LLM_API_KEY and optionally SOULPRINT_LLM_BASE_URL. Note: cloud providers send conversation excerpts to external servers. 

# OpenAI direct
SOULPRINT_LLM_PROVIDER=openai
SOULPRINT_LLM_API_KEY=sk-...

# Anthropic
SOULPRINT_LLM_PROVIDER=anthropic
SOULPRINT_LLM_API_KEY=sk-ant-...
