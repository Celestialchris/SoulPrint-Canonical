The product loop
01

Import

Drop your ChatGPT .zip, Claude .json, or Gemini Takeout. Provider auto-detected. Normalized into one canonical ledger.

02

Browse

Workspace overview, transcript explorer with prompt-level TOC and minimap, federated cross-provider view.

03

Search & Ask

Full-text across all providers. Grounded answering that cites your specific conversations. Every answer has an auditable trace.

04

Discover

Cross-conversation topics. Per-conversation summaries. Multi-conversation digests. All derived, all traceable.

05

Continue

Generate a compact continuity packet from any conversation. Copy a structured bridge into your next chat.

06

Export

Memory Passport with manifest, canonical JSONL lanes, provenance index, and checksums. Verifiable.

From the manifesto
What SoulPrint is not
—

Not a hosted SaaS. Your data never leaves your machine.

—

Not a mem0 clone. SoulPrint is for users, not developer infrastructure.

—

Not an AI dashboard. No metrics theater. No admin-panel energy.

—

Not a generic wrapper. SoulPrint has its own canonical ledger and trust chain.

Architecture
TruthCanonical SQLite ledger. Explicit lanes. Stable provenance.
LegibilityBrowse, search, inspect, trace, export. Read-only over truth.
IntelligenceSummaries, topics, digests, continuity. All derived. All traceable.
DistributionDesktop app, landing page, freemium gate.

Every derived artifact stores source conversation IDs, generation timestamp, LLM provider, and prompt template version. Derived never impersonates canonical.

Quick start
# Clone and run in 60 seconds
git clone https://github.com/Celestialchris/SoulPrint-Canonical.git
cd SoulPrint-Canonical

pip install -r requirements-minimal.txt
python -m src.run

# Open http://127.0.0.1:5678
# Drop an export file on the Import page

Intelligence features (summaries, topics, ask) use your own API key. Without a key, import, browse, search, and export all work fully.
