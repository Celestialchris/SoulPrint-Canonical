# Wrapped Summary Page — Design Spec

**Date:** 2026-03-17
**Status:** Approved
**Task:** Task 3 from SoulPrint Execution Plan

## Purpose

SoulPrint's "Spotify Wrapped for AI." A standalone cinematic page users screenshot and share. Always dark Torchlit Vault. Strongest glow surface in the product. Free tier — no license gate.

## Architecture

### Viewmodel: `src/app/viewmodels/wrapped.py`

Frozen dataclass `WrappedSummary` + `build_wrapped_summary()` builder function. Same pattern as `WorkspaceSummary`. Pure read-only queries against canonical SQLite ledger + derived JSONL stores.

#### Fields

| Field | Source | Type |
|-------|--------|------|
| total_conversations | imported count + native count | int |
| total_messages | imported messages + native entries | int |
| providers | GROUP BY source with percentage | list[dict] |
| dominant_provider | max from providers | dict |
| date_range | min/max timestamps from conversations | dict |
| most_active_month | GROUP BY year-month on created_at_unix | str |
| longest_conversation | max message count per conversation | dict |
| topic_highlights | top 5 from latest topic scan JSONL | list[str] |
| average_messages_per_conversation | total_messages / total_conversations | float |
| unfinished_threads | two-tier: continuity open_loops + heuristic | dict |
| has_data | bool — gates empty state | bool |

### Unfinished Threads — Two-Tier Detection

1. Check continuity store for `open_loops` artifacts → extract conversation titles from `source_conversation_ids`
2. Fallback: conversations where last message `role == "user"` (unanswered question)
3. Deduplicate by conversation ID, cap at top 3 titles

### Route: `GET /summary`

Added in `create_app()`. Returns `wrapped.html` with `WrappedSummary` viewmodel. Reads topic store and continuity store paths from SQLite URI.

### Post-First-Import Redirect

In the `POST /import` handler: capture `ImportedConversation.query.count()` before import. If was 0 and now >0 → `redirect("/summary")`. If incremental (was >0) → stay on import page, add `show_summary_link: True` to result context.

### Template: `src/app/templates/wrapped.html`

Standalone HTML — does NOT extend `base.html`. Own `<head>` with Google Fonts link, all CSS inline. Always dark. No nav, no sidebar. Full viewport, max-width ~800px centered.

#### Sections (top to bottom)

1. **Empty state** (if no data): wordmark + "Import your first conversation" + wine CTA → `/import`
2. **Hero**: wordmark with ember glow text-shadow + "Your AI Memory" + date range
3. **Headline stat**: total messages, Forum `clamp(4rem,10vw,8rem)`, gold color
4. **Supporting stats**: 3-column grid (provider count, avg messages, total conversations)
5. **Provider breakdown**: horizontal bars with lane colors (ChatGPT green, Claude gold, Gemini blue)
6. **Most active month + longest conversation**
7. **Topics** (conditional — skip entirely if no topic data exists)
8. **Unfinished threads** (always show, even if count is 0; "Continue one" link if threads exist)
9. **Watermark**: "Generated by SoulPrint" + "soulprint.dev"
10. **Footer**: "Back to Workspace" link + screenshot encouragement text

#### Visual Spec

- Background: `#0a0908` (darker than standard Torchlit Vault `#0e0d0b`)
- Layered cinematic gradients + grain overlay at 0.018 opacity
- Vignette: vertical gradient fading to black at top/bottom
- Wordmark glow: five-layer text-shadow per brand.md "Embers in Velvet Darkness"
- Typography: Forum for wordmark/headline, Cormorant Garamond for body, JetBrains Mono for stats
- Hierarchy: `--t1` through `--t4` opacity scale
- Wine `#6b3a3a` for CTAs only, gold `#c9a84c` / `#a08848` for headings/stats
- No cards, no shadows, no border-radius
