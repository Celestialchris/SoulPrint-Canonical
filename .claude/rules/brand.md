# Brand Rules (USB Drive)
- Dark default: --bg #0e0f11, --surface #141518, --raised #1a1b1f
- Accents: --accent #4ade80 (CTAs/actions), --accent-secondary #7c5cbf (interactive feedback)
- Text hierarchy: --t1 rgba(240,242,238,0.88), --t2 0.55, --t3 0.35, --t4 0.15
- Fonts: Forum (wordmark only), system sans-serif (body), JetBrains Mono (mono)
- Lane stripes: ChatGPT #4ade80, Claude #a78bfa, Gemini #60a5fa, Native #60a5fa
- No icons in nav. No weight > 500.
- Green = safe, verified, go. Purple = you're here, interactive feedback.
- Never use: vault, ember, torchlit, wine, gold accent, Cormorant Garamond
